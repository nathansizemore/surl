# surl

Converts Spotify share links from URL to URI.

---

```
surl - Converts Spotify share links from URL to URI

Usage:
    surl <url>
    surl (-h | --help)
    surl --version

Options:
    -h --help       Show this screen.
    --version       Show version.
```
